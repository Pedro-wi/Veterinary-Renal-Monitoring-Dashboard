# Veterinary Renal Monitoring Dashboard

This project simulates a renal monitoring dashboard for veterinary patients,
with focus on creatinine and urea visualization to support clinical reasoning.

## Motivation
Renal disease monitoring in veterinary medicine often lacks simple tools
for visualizing laboratory trends over time.

## Technologies
- Python
- Pandas
- Streamlit
- Matplotlib

## How to run
pip install -r requirements.txt
streamlit run app.py
